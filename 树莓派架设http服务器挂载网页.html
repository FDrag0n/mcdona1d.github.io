树莓派架设http服务器挂载网页

Date 2015-01-02 19:04 Category tech Tags RaspberryPi / Apache

需要在树莓派上挂在一个小型的网页测试用，比较方便 因为有之前架设Wordpress的经验，在树莓派上就简单许多

我们要用到的软件就是apache2

安装apache2

sudo apt-get install apache2

配置

因为我的80端口被微信公众平台占着，所以我不能让网页走80端口，需要更改端口

nano /etc/apache2/ports.conf

将Listen 80改为Listen 8080

重启服务

service apache2 restart

然后就可以在浏览器上先测试一下，输入http://你的树莓派局域网ip:8080/尝试一下能否访问了 如果显示一下界面，apache2就正常工作了

接下来我们就可以把我们想要挂在的网页上传到/var/www目录下了，替换index.html
