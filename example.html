0. 什么是 DMA

所谓的 Direct memory access（DMA），是指让外设能够不经过 CPU 而直接访问内存。其目的是让 CPU 和外设之间交换数据。相比于 CPU 主动去读总线，然后把数据放进内存，其好处在于传输速度比较快，外设读写内存期间 CPU 是闲着的，可以做些别的事情。DMA 核心就是 DMA 控制器和总线，简单来说，DMA 控制器负责操作总线，将数据写入内存，或者将数据从内存读出。
